# explorer-gps [![CircleCI](https://circleci.com/gh/CMUCloudComputing/explorer-gps.svg?style=svg)](https://circleci.com/gh/CMUCloudComputing/explorer-gps)
NodeJS GPS service for explorer

## Development
### Installation
- `cd explorer-gps`
- `npm install`

### Running the server
- `npm run start`

### Running Tests
- `npm run test`

### Linting
- `npm run lint`
